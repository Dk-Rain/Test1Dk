# Task 1 – Collapsible Sidebar (Test 1 DK)

## Project Overview
This project was completed as Task 1 of the Elevvo Pathways Front-End Web Development Internship. The goal of the task was to build a collapsible and responsive sidebar using core frontend technologies. The sidebar can be opened and closed using a toggle button and adapts well to both mobile and desktop screen sizes.

## Features
- Collapsible sidebar with toggle button
- Responsive design for mobile and desktop screens
- Smooth open and close animation
- Logo placeholder at the top of the sidebar
- Navigation menu with icons
- Clean and simple layout

## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome (for icons)

## How It Works
- The toggle button controls the visibility of the sidebar using JavaScript.
- CSS transitions are used to animate the sidebar when opening and closing.
- Media queries ensure the sidebar behaves correctly on different screen sizes.

## How to Run the Project
1. Download or clone the repository.
2. Open index.html in any modern web browser.
3. Click the menu button to open or close the sidebar.
